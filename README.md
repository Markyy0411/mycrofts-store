## Professional Summary
Mycroft's Store is a modern, responsive e-commerce landing page designed for selling digital assets, gaming credits, and premium streaming subscriptions. The project features a dark-themed UI, smooth animations, and optimized layout for various devices, providing a seamless user experience.

## 🛠️ Core Competencies Demonstrated
*   **Web Development**: Proficient in HTML5 and CSS3 for creating structured and responsive layouts.
*   **User Interface Design**: Implementation of modern dark-mode aesthetic with interactive, glowing elements.
*   **Performance & Responsiveness**: Mobile-first design principles using CSS Grid and media queries.
*   **User Experience (UX)**: Integration of intuitive navigation, clear call-to-action (CTA) buttons, and persistent communication channels.

## 📂 Project Structure & Features
- `index.html`: Main landing page with sections for gaming, streaming, and productivity tools.
- `CSS Styling`: Embedded CSS providing custom animations (pulseGlow, borderPulse), responsive grid system, and layout management.
- `Interactive Elements`: Floating chat button for direct customer engagement and hover-responsive service cards.

## 🚀 Getting Started
1. Clone the repository: `git clone https://github.com/Markyy0411/mycrofts-store.git`
2. Open `index.html` in your preferred web browser to view the store.
